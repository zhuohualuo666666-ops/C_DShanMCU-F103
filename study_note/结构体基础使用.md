1.strcpy函数的作用：
char names[3][20];
strcpy(names[0],"zhangsan");

strcpy 从 "zhangsan" 的起始地址开始逐字节解引用其内容，
尝试将其写入 names这个多维数组的第0行指向的20列 (字节)的内存空间。
如果包含\0在内的字符串长度大于二十会发生缓冲区溢出.


2结构体：

struct Student{
    char name[20];
    int age;
    char sex;
    int score;
    char lastnameinitial;
    char firstnameinitial;
    char sexualpreference;
};

这个结构体在分配内存时sex这个char类型的变量分配1字节的空间，
在分配下一个int类型的变量age时,为了内存对齐使int类型变量score放在4的倍数地址上，
导致了3个字节内存的浪费。

struct Student{
    char name[20];
    int age;
    char sex;
    char lastnameinitial;
    char firstnameinitial;
    char sexualpreference;
    int score;
};

将这四个char类型的成员连续放在一起，
可以有效避免之前那 3 字节的填充浪费。
