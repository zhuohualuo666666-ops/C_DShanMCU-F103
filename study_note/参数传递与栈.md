<!-- 参数传递与栈 -->


void change_val(int a)
{
    printf("change_val 1 a=%d\r\n",a);
    a=200;
    printf("change_val 2 a=%d\r\n",a);
}

void change_val2(int *a)
{
    printf("change_val 1 a=%d\r\n",*a);
    *a=200;
    printf("change_val 2 a=%d\r\n",*a);
}

int mymain(void)
{
    int a=100;
    change_val(a);
    printf("mymain after change_val,a=%d\r\n",a);
    change_val2(&a);
    printf("mymain after change_val2,a=%d\r\n",a);
    return 0;
}


第一个change_val函数修改的是调用该函数时开辟的内存空间里的a的值，
与mymain函数里第一句定义的局部变量a的值不是同一个；

mymain函数里第一句执行完，栈指针sp由高地址向低地址移动，在内存中开辟了一块空间存放mymain函数的局部变量a的值，我用“a1”指代它，
调用change_val函数，栈指针sp由高地址向低地址移动，内存中再开辟了一块空间存放change_val函数的局部变量a的值，我用“a2”指代它，
change_val函数执行完，栈指针sp由低地址向高地址移动，回收change_val函数开辟出的栈空间，
该函数改变的是“a2”的值，“a1”的值并没有改变，
printf("mymain after change_val,a=%d\r\n",a)这一句打印的是“a1”的值，仍是100；

mymain函数调用change_val2函数，栈指针sp由高地址向低地址移动，内存中再开辟了一块空间存放“a1”的地址的值，
change_val2函数通过“a1”的地址的值去访问4个字节，修改“a1”的值，写入200
printf("mymain after change_val2,a=%d\r\n",a)这一句打印的还是“a1”的值，已被修改为200



change_val是值传递，
change_val2是地址传递，
两者本质上都是值传递，
地址传递本质上是把地址的值作为参数传进去
