 <!-- 指针数组 -->
1.定义：
int *arr[3];
意思是3个连续 int *类型的变量，内存中开辟了12个字节的空间

int a=1;
int b=2;
int c=3;


2.引用：
arr[0]=&a;
arr[1]=&b;
arr[2]=&c;
其中，arr[0]是一个int *类型的变量，可以保存int类型变量的地址。
执行完这三句后，引用时arr[1]等同于&b
*arr[1]等同于b

3.额外
char *p="ABCDERF";
这一句定义将ABCDERF这个字符串首字符的地址赋给了char *类型的指针变量p


char *strs[] = {"Hello", "World", "C Programming"};
第一句话的意思是将Hello这个字符串的首地址赋给char＊类型的指针数组strs的第0项，
World这个字符串的首地址赋给char＊类型的指针数组strs的第1项，
C Programming这个字符串的首地址赋给char＊类型的指针数组strs的第2项

printf("%s\n\r", strs[0]); 
这里打印出的效果是Hello这个字符串，因为strs[0] 本身是地址，
而%s要求接收一个char *类型的值，
即 printf 函数内部对这个地址有解引用的操作。
