·main.c文件：


int g_main_a=0x12345678;

·test2.c文件：


static int m=0x5555;   //静态全局变量

void change_val(int a)
{
    printf("change_val 1 a=%d\r\n",a);
    a=200;
    printf("change_val 2 a=%d\r\n",a);
}

void change_val2(int *a)
{
    printf("change_val 1 a=%d\r\n",*a);
    *a=200;
    m++;//对于定义在前面的全局变量可以用，若是m在该语句后面调用编译器会报错，优先使用自己文件定义的全局变量（作用域更小的），0x5555加上1而不是123加上1
    printf("change_val 2 a=%d\r\n，m=0x%x\r\n",*a,m);
}

int factorial(int n)
{
    static int cnt=0;//静态局部变量，只分配一次，初始化一次
    cnt++;
    if(0==n)
    {
    printf("call factorial time=%d\r\n",cnt);//打印调用次数
    return 1;
    }
    return n * factorial(n-1);
}

int a=0x1111;

int mymain(void)
{
    int a=0xaaaa;//局部变量
    extern int g_main_a;//调用其他文件的变量时，需要在前面加上extern,编译器才知道从外部文件找
    printf("g_main_a=0x%x\r\n",g_main_a);
    printf("before change_val2:a=0x%x,m=0x%x\r\n",a,m); //优先使用局部变量，a打印出来值为aaaa
    change_val2(&a);
    printf("after change_val2:a=0x%x,m=0x%x\r\n",a,m);
    factorial(3);//打印出来调用次数为4次
    return 0;
}



·理解：

局部变量作用域为定义它的函数里，只能在该函数中被调用；
全局变量作用域为全部文件，定义后当前文件里后面可以调用，
若要在其他文件中调用，需要声明是外部变量，加上extern。

如果在多个文件里定义了相同的变量名，编译器会报multiply define的错误，可以加上static限制作用域范围；
两个文件都定义int m,编译器会报multiply define的错误;
一个是int m，另一个是static int m，则不会报错，static将变量限制在文件内部互不干扰

全局变量加上static后，只能在定义它的文件里使用，即作用域缩小为当前文件；
局部变量加上static后，变为静态局部变量，只分配一次，初始化一次。

普通局部变量运行到的时候临时、动态分配；
静态局部变量，只分配一次，初始化一次。

定义了同名的其他文件全局变量和本文件静态全局变量，调用时优先使用本文件的全局变量；
本文件中定义了同名的全局变量和局部变量，局部变量所在函数调用时优先使用局部变量；
一句话：优先使用作用域更小的。

普通局部变量存在栈里；
静态局部变量、全局变量存在数据段中，
初始化的存在.data段，未初始化的存在.bss段
