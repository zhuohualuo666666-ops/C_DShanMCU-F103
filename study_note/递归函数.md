<!-- 递归函数 -->

int factorial(int n)
{
    if(0==n)
    {
    return 1;
    }
    return n * factorial(n-1);
}

int mymain(void)
{
    int n=3;
    int f=factorial(n);
    printf("f=%d\r\n",f);
    return 0;
}

mymain函数调用factorial函数时，开辟一块栈空间，
执行return n * factorial(n-1)时需要再次调用factorial函数，再次开辟栈空间，
直到最后n=0时最后一次调用的factorial函数返回1，
此时最外层调用的factorial函数返回最终计算的阶乘值，一层层回收栈空间。
因此如果递归调用很深，递归函数容易造成栈溢出。递归函数通常会有一个退出机制，防止无限递归。


递归函数可以理解为自己嵌套调用自己
