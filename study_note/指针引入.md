<!-- 指针引入 -->


&为取地址符号
*在声明语句里是指变量类型为指针，在执行语句里是指解引用



int a = 13;  // 0x0000000d
    int b = 3;
    
    char c = 'A';
    
    char * addr = &c; //char类型的指针可以存入char类型变量的地址，类型要对应
    int * addr2 = &a; //int类型的指针可以存入int类型变量的地址，类型要对应
    
    printf("a's address = 0x%x\n\r", &a);
    printf("b's address = 0x%x\n\r", &b);
    printf("c's address = 0x%x\n\r", &c);
    
    printf("addr = 0x%x\n\r", addr);
    printf("addr's address = 0x%x\n\r", &addr);

    printf("addr2 = 0x%x\n\r", addr2);
    printf("addr2's address = 0x%x\n\r", &addr2);
    
    printf("sizeof(char *) = %d\n\r", sizeof(char *));
    printf("sizeof(short *) = %d\n\r", sizeof(short *));
    printf("sizeof(int *) = %d\n\r", sizeof(int *));
    printf("sizeof(long *) = %d\n\r", sizeof(long *));


这个案例内存中，变量a，b是int类型，占据4个字节；

变量c是char类型，占据1个字节；

addr这个变量是一个指针变量，stm32中地址是32位的，
因此指针变量存储空间也是32位，即占据4个字节，里面存放变量c的地址；

&addr意思是指针变量addr的地址，程序运行到这一步，相当于对变量c取地址后又取了它地址的地址；
