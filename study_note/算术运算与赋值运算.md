<!-- 算数运算与赋值预算 -->


算术运算规则：做算术运算时，如果两个操作数中有一个是浮点类型，另一个整数类型会自动转换为浮点类型。


整数与整数运算结果也是整数：
int a=13;
int b=3;
int c;
c=a/b;
则c的值为4；

浮点数与整数相除结果是浮点数：
float a=13;
int b=3;
float c;
c=a/b;
则c的值为4.333333...

整数与浮点数相除结果是浮点数：
int a=13;
float b=3;
float c;
c=a/b;
则c的值为4.333333...

隐式转换（运算顺序影响结果）：
int a=13;
int b=3;
float c;
c=1.0*a/b;
则c的值为4.333333...

int a=13;
int b=3;
float c;
c=a/b*1.0;
则c的值为4.0000

强制转换：
int a=13;
int b=3;
float c;
c=(float)a/b;
则c的值为4.333333...
